# Labels Reference

Select **1 Type label** for PRs. Priority labels are issue-only.

## Type Labels (pick 1)

| Label | Description | When to use |
|-------|-------------|-------------|
| `✨ Feature` | 신규 기능 | New feature |
| `♻️ Refactor` | 코드 리팩토링 | Refactoring |
| `🐞 Bug` | 버그 발생 | Bug fix |
| `📝 Docs` | 문서 추가/변경 | Docs, README |
| `✅ Test` | Test 관련 사항 | Test code |
| `⚙ Setting` | 환경 세팅 | Config, CI/CD, build |
| `⚡️ Simple` | 간단한 변경사항 | Minor changes |
| `⚡️performance` | 성능 개선 | Performance improvement |
| `🌏 Deploy` | 배포 관련 | Deployment |

## Off-limits Labels (do NOT assign)

`Priority: *`, `🙋‍♂️ Question`, `🪡 Want` are issue-only — never assign to PRs.

## Quick Decision

```
New feature?              → ✨ Feature
Refactoring?              → ♻️ Refactor
Bug fix?                  → 🐞 Bug
Docs, README?             → 📝 Docs
Test code?                → ✅ Test
Config, CI/CD, build?     → ⚙ Setting
Minor change?             → ⚡️ Simple
Performance improvement?  → ⚡️performance
Deployment?               → 🌏 Deploy
```
